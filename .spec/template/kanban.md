# カンバンボード

「カンバンボード」はタスクの状態を視覚化する管理ツールです。

- **Backlog**: 「未着手」の状態を表す。
- **In-Progress**: 「着手中」の状態を表す。
- **Done**: 「完了」の状態を表す。
